Introduction à GTK Xfce Engine

Le paquet GTK Xfce Engine contient plusieurs thèmes GTK+ 2 et GTK+ 3 et les bibliothèques nécessaires pour les afficher. Cela sert à personnaliser l'apparence de votre bureau Xfce.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de GTK Xfce Engine

Installez GTK Xfce Engine en lançant les commandes suivantes : 

sed -i 's/\xd6/\xc3\x96/' gtk-3.0/xfce_style_types.h &&
./configure --prefix=/usr --enable-gtk3              &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

sed -i 's/\xd6/\xc3\x96/' ... : Le script glib-mkenums a été réécrit de perl en python et crashera en rencontrant un caractère ISO-8859-1 dans un commentaire. Ce sed convertit ce caractère en UTF-8.
