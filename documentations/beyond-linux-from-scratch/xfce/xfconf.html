Xfconf-4.12.1

Introduction à Xfconf

Xfconf est le système de stockage de configuration pour Xfce.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://archive.xfce.org/src/xfce/xfconf/4.12/xfconf-4.12.1.tar.bz2

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 20dc8d2bfd80ba136bf4964021b32757

  • Taille du téléchargement : 552 Ko

  • Estimation de l'espace disque requis : 7.4 Mo

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de Xfconf

Requises

dbus-glib-0.110 et libxfce4util-4.12.1

Facultatives

GTK-Doc-1.29 et le module perl Glib (cela permet aux développeurs perl de stocker des paramètres dans xfconf pour les applications perl)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/xfconf

Installation de Xfconf

Installez Xfconf en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--enable-gtk-doc : Utilisez ce paramètre si GTK-Doc est installé et si vous souhaitez reconstruire et installer la documentation de l'API.

Contenu

Programme installé: xfconf-query
Bibliothèque installée: libxfconf.so
Répertoires installés: /usr/include/xfconf-0, /usr/lib/xfce4/xfconf et /usr/share/gtk-doc/html/xfconf

Descriptions courtes

xfconf-query

est un outil en ligne de commande pour voir ou modifier un paramètre stocké dans xfconf.

libxfconf.so

contient les fonctions de base de la configuration de Xfce.
