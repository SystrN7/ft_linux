Introduction à Xfce4 Keyboard Plugin

Si votre système a un clavier standard (US), vous n'avez probablement pas besoin d'installer ce greffon.

Le paquet de greffon de clavier Xfce4 contient un greffon qui ajoute un écran de Xfce4 très personnalisable et pratique pour initialiser et utiliser plusieurs claviers (actuellement jusqu'à 4 à cause d'une limite du protocole X11) et passer de l'un à l'autre, individuellement pour chaque application, individuellement pour chaque fenêtre, ou globalement pour le bureau et toutes les applications et fenêtres.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de Xfce4 Keyboard Plugin

Installez Xfce4 Keyboard Plugin en lançant les commandes suivantes : 

sed -e 's|xfce4/panel-plugins|xfce4/panel/plugins|' \
    -i panel-plugin/{Makefile.in,xkb-plugin.desktop.in.in} &&

./configure --prefix=/usr         \
            --libexecdir=/usr/lib \
            --disable-debug       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

sed -e ... xkb-plugin.desktop.in.in} et --libexecdir=/usr/lib : Le sed et le paramètre sont nécessaires pour placer xkb-plugin dans le répertoire par défaut des greffons de Xfce4.

--disable-debug : Ce paramètre désactive le support de debogage.
