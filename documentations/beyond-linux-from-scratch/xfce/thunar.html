Thunar-1.8.4

Introduction à Thunar

Thunar est le gestionnaire de fichiers Xfce, une interface graphique Gtk+ 3 pour organiser les fichiers de votre ordinateur.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Dépendances de Thunar

Requises

Exo-0.12.4 et libxfce4ui-4.12.1

Requises (à l'exécution)

gnome-icon-theme-3.12.0 ou lxde-icon-theme-0.5.1

Recommandées

Facultatives

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/thunar

Installation de Thunar

Installez Thunar en lançant les commandes suivantes : 

./configure --prefix=/usr \
            --sysconfdir=/etc \
            --docdir=/usr/share/doc/Thunar-1.8.4 &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contenu

Programmes installés: Thunar, thunar et thunar-settings
Bibliothèques installées: libthunarx-3.so et /usr/lib/xfce4/panel/plugins/libthunar-tpa.so
Répertoires installés: /etc/xdg/Thunar, /usr/include/thunarx-3, /usr/lib/{Thunar,thunarx-3}, /usr/share/doc/Thunar-1.8.4, /usr/share/gtk-doc/html/thunarx et /usr/share/{pixmaps,}/Thunar

Descriptions courtes

Thunar

est un lien symbolique vers thunar.

thunar

est le gestionnaire de fichiers de Xfce.

thunar-settings

est un script shell qui lance une boîte de dialogue pour vous permettre de modifier le comportement de Thunar.

libthunarx-3.so

contient la bibliothèque d'extension Thunar qui permet d'ajouter de nouvelles fonctionnalités au gestionnaire de fichiers Thunar.
