Ark-18.12.2

Introduction à Ark

Le paquet Ark est un outil d'archivage de KF5. Il s'agit d'une interface graphique à tar et des outils similaires.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://download.kde.org/stable/applications/18.12.2/src/ark-18.12.2.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 7f5229e7003df3dc66f5f7b1e928b950

  • Taille du téléchargement : 2.5 Mo

  • Estimation de l'espace disque requis : 25 Mo

  • Estimation du temps de construction : 0.4 SBU (avec parallélisme = 4)

Dépendances de Ark

Requises

KDE Frameworks-5.55.0 et libarchive-3.3.3

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/ark5

Installation de Ark

Installez Ark en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contents

Programmes installés: ark et arkprofile
Bibliothèques installées: libkdeinit5_ark.so et libarkprivate.so
Répertoires installés: $KF5_PREFIX/share/doc/HTML/*/ark, $KF5_PREFIX/share/ark et $KF5_PREFIX/share/kxmlgui5/ark

Descriptions courtes

ark

est un émulateur de terminal X qui fournit une interface en ligne de commande.

arkprofile

est un outil en ligne de commande qui change les options de profil de l'onglet actuel.
