Phonon-backend-vlc-0.10.2

Introduction à Phonon-backend-vlc

Ce paquet fournit un moteur Phonon qui utilise le framework média VLC.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://download.kde.org/stable/phonon/phonon-backend-vlc/0.10.2/phonon-backend-vlc-0.10.2.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : fb25f7b12e04a62d7d1916bade88400f

  • Taille du téléchargement : 60 Ko

  • Estimation de l'espace disque requis : 2.9 Mo

  • Estimation du temps de construction : 0.2 SBU

Dépendances de Phonon-backend-vlc

Requises

phonon-4.10.2 et VLC-3.0.6

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/phonon-backend-vlc

Installation de Phonon-backend-vlc

Installez Phonon-backend-vlc en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr    \
      -DCMAKE_BUILD_TYPE=Release     \
      -DPHONON_BUILD_PHONON4QT5=ON   \
      -Wno-dev .. &&
make

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DCMAKE_BUILD_TYPE=Release : Ce paramètre est utilisé pour appliquer un plus haut niveau d'optimisation du compilateur.

-DPHONON_BUILD_PHONON4QT5=ON : Ce paramètre est utilisé pour assurer que la version Qt5 est construite même si Qt4 est présent.

Contenu

Programme installé: aucun
Bibliothèques installées: phonon_vlc.so
Répertoire installé: /usr/lib/qt5/plugins/phonon4qt5_backend
