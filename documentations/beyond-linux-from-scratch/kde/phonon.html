Phonon-4.10.2

Introduction à Phonon

Phonon est l'API multimédia de KDE4. Il remplace le vieux aRts, qui n'est plus supporté par KDE. Phonon nécessite également le moteur GStreamer ou VLC.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

Dépendances de Phonon

Requises

CMake-3.13.4, extra-cmake-modules-5.55.0, GLib-2.58.3 et Qt-5.12.1

Au moins l'un des paquets phonon-backend-gstreamer-4.9.0 ou phonon-backend-vlc-0.10.2 doit être installé après pour les opérations multimédia dans KDE.

Facultatives

PulseAudio-12.2

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/phonon

Installation de Phonon

Installez Phonon en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr    \
      -DCMAKE_BUILD_TYPE=Release     \
      -DPHONON_BUILD_PHONON4QT5=ON   \
      -Wno-dev .. &&
make

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DCMAKE_BUILD_TYPE=Release : Ce paramètre est utilisé pour appliquer un plus haut niveau d'optimisation du compilateur.

-DPHONON_BUILD_PHONON4QT5=ON : Ce paramètre est utilisé pour s'assurer que la version Qt5 de la bibliothèque est construite même si Qt4 est présent.

Contenu

Programmes installés: Aucun
Bibliothèques installées: libphonon4qt5.so libphonon4qt5experimental.so
Répertoires installés: /usr/include/phonon4qt5, /usr/lib/cmake/phonon4qt5 et /usr/share/phonon4qt5
