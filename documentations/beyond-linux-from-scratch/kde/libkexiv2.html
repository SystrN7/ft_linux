libkexiv2-18.12.2

Introduction à libkexiv2

Libkexiv2 est une enveloppe KDÊ autour de la bibliothèque Exiv2 pour manipuler les métadonnées des images.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://download.kde.org/stable/applications/18.12.2/src/libkexiv2-18.12.2.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 1d1dd42a07f377aab1c2443f716160fc

  • Taille du téléchargement : 64 Ko

  • Estimation de l'espace disque requis : 3.3 Mo

  • Estimation du temps de construction : 0.2 SBU

Dépendances de libkexiv2

Requises

KDE Frameworks-5.55.0 et Exiv2-0.27.0

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/libkexiv2

Installation de libkexiv2

Tout d'abord corrigez plusieurs fichiers pour que Exiv2-0.27.0 soit correctement trouvé et utilisé : 

patch -Np1 -i ../libkexiv2-18.12.2-exiv2_0.27-1.patch

Installez libkexiv2 en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
