Kdenlive-18.12.2

Introduction à Kdenlive

Le paquet Kdenlive est un éditeur vidéo basé sur KF5.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://download.kde.org/stable/applications/18.12.2/src/kdenlive-18.12.2.tar.xz

  • Estimation de l'espace disque requis : 120 Mo

Dépendances de Kdenlive

Requises

KDE Frameworks-5.55.0 mlt-6.12.0 et v4l-utils-1.16.3 (exécution)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/kdenlive5

Installation de Kdenlive

Tout d'abord corrigez des problèmes identifiés par gcc7 : 

sed -e '/KLocal/a #include <functional>' \
    -i src/profiles/tree/profiletreemodel.cpp  &&

sed -e '/abs/s/leftDist/(int)&/' \
    -i src/scopes/audioscopes/spectrogram.cpp

Installez Kdenlive en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contenu
