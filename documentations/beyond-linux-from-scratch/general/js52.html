Introduction à JS

JS est le moteur de JavaScript écrit en C par Mozilla. Ce paquet est présent pour GJS et polkit.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de JS

Installez JS en lançant les commandes suivantes :

Note

If installing in the chroot environment, be sure to export the SHELL environment variable or configuration of the package will fail. 

cd js/src &&

./configure --prefix=/usr       \
            --with-intl-api     \
            --with-system-zlib  \
            --with-system-nspr  \
            --with-system-icu   \
            --enable-threadsafe \
            --enable-readline   &&
make

Ce paquet ne contient pas de suite de tests utilisable.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--with-* : Ces paramètres permettent au système de construction d'utiliser les versions du système des bibliothèques présentées. Elles sont nécessaires pour la stabilité.

--enable-readline : Ce paramètre active le support de Readline dans le shell JS.

--enable-threadsafe : Ce paramètre active le support pour les threads multiples. Cela améliore les performances ainsi que la stabilité.

Descriptions courtes

js52

fournit une interface en ligne de commande au moteur JavaScript.

js52-config

est utilisé pour trouver le compilateur JS et les drapeaux de l'éditeur de liens

libmozjs-52.so

contient les fonctions de l'API JavaScript de Mozilla.
