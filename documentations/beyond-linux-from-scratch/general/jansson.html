Jansson-2.12

Introduction à Jansson

Le paquet Jansson contient une bibliothèque utilisée pour encoder, décoder et manipuler des données JSON.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://www.digip.org/jansson/releases/jansson-2.12.tar.gz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 0ed1f3a924604aae68067c214b0010ef

  • Taille du téléchargement : 480 Ko

  • Estimation de l'espace disque requis : 7.3 Mo

  • Estimation du temps de construction : 0.1 SBU

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/jansson

Installation de Jansson

Installez jansson en lançant les commandes suivantes : 

./configure --prefix=/usr --disable-static &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install
