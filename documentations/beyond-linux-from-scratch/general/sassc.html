sassc-3.5.0

Introduction à sassc

SassC est une enveloppe autour de libsass, un langage de prétraitement de CSS.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de sassc

Tout d'abord, construisez la bibliothèque : 

autoreconf -fi &&

./configure --prefix=/usr --disable-static &&
make

Ce paquet n'a pas de suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Construisez l'enveloppe en ligne de commande : 

tar -xf ../sassc-3.5.0.tar.gz &&
cd sassc-3.5.0 &&
autoreconf -fi &&

./configure --prefix=/usr &&
make

Ce paquet n'a pas de suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
