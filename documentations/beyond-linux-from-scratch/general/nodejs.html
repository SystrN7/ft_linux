Node.js-10.15.1

Introduction à Node.js

Node.js est un programme JavaScript construit sur le moteur JavaScript V8 de Chrome's.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : https://nodejs.org/dist/v10.15.1/node-v10.15.1.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 1cad7963255de53509bfa560221bdc88

  • Taille du téléchargement : 19 Mo

  • Estimation de l'espace disque requis : 426 Mo (plus 51 Mo pour les tests)

  • Estimation du temps de construction : 5.2 SBU (avec parallélisme = 4 ; plus 1.2 SBU pour les tests)

Note

Cette archive a été créée avec une version BSD de tar et l'extraire avec une version linux produira des avertissements sans conséquence à propos de mots-clefs d'en-têtes étendus inconnus.

Installation de Node.js

Construisez Node.js en lançant les commandes suivantes : 

./configure --prefix=/usr                  \
            --shared-cares                 \
            --shared-libuv                 \
            --shared-nghttp2               \
            --shared-openssl               \
            --shared-zlib                  \
            --with-intl=system-icu         &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install &&
ln -sf node /usr/share/doc/node-10.15.1

Explication des commandes

--with-intl=small-icu : utilisez la version système de icu. Les autres valeurs sont icu-full (pour construire une version complète locale de la bibliothèque icu) et system-icu (pour construire une version minimale locale de la bibliothèque icu).

--shared-{cares,libuv,nghttp2,openssl,zlib} : utilise les bibliothèques du système au lieu des copies locales.

--without-npm : ne construit pas npm (utilisez cette option si vous souhaitez construire npm séparément plus tard).

--shared-http-parser : utilise la bibliothèque installée sur le système plutôt qu'une copie locale.

Contents

Programmes installés: node, npm et npx
Bibliothèque installée: None
Répertoires installés: /usr/include/node/ et /usr/lib/node_modules/npm/

Descriptions courtes

node

est l'exécutable JavaScript côté serveur.

npm

est le gestionnaire de paquet de Node.js.

/usr/lib/node_modules/npm/

est la racine de l'installation pour les exécutables Node.js et ses bibliothèques.
