libuv-1.26.0

Introduction à libuv

Le paquet libuv est une bibliothèque de support multi-plateforme qui se concentre sur les entrées-sorties asynchrones.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : https://dist.libuv.org/dist/v1.26.0/libuv-v1.26.0.tar.gz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 3d0537a24f79e6c89333de949f8bfe84

  • Taille du téléchargement : 1.1 Mo

  • Estimation de l'espace disque requis : 13 Mo (plus 11 Mo pour les tests)

  • Estimation du temps de construction : 0.1 SBU (plus 0.4 SBU pour les tests)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/libuv

Installation de libuv

Installez libuv en lançant les commandes suivantes : 

sh autogen.sh                              &&
./configure --prefix=/usr --disable-static &&
make

Si vous voulez lancer les tests, lancez make check.

Maintenant, en tant qu'utilisateur root : 

make install
