SCons-3.0.4

Introduction à SCons

SCons est un outil pour construire des logiciels (et d'autres fichiers) implémenté en Python.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : https://downloads.sourceforge.net/scons/scons-3.0.4.tar.gz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : dcf8ed13acde28040784606bb3d5fb84

  • Taille du téléchargement : 640 Ko

  • Estimation de l'espace disque requis : 5.4 Mo

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de SCons

Facultatives

docbook-xsl-1.79.2, libxslt-1.1.33

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/scons

Installation de SCons

Installez SCons en lançant les commandes suivantes en tant qu'utilisateur root : 

sed -i 's/env python/&3/' script/scons*            &&
python3 setup.py install --prefix=/usr  \
                         --standard-lib \
                         --optimize=1   \
                         --install-data=/usr/share &&
rm -v /usr/bin/scons*.bat

Contenu

Programmes installés: scons, scons-configure-cache, sconsign et scons-time
Bibliothèques installées: Aucune
Répertoire installé: /usr/lib/python3.7/site-packages/SCons

Descriptions courtes

scons

est un outil de construction logiciel.

scons-configure-cache

affiche ou convertit la configuration d'un répertoire de cache de SCons.

sconsign

affiche les informations du fichier .sconsign de SCons.

scons-time

génère et affiche les informations de temps de SCons.
