GLibmm-2.58.0

Introduction à GLibmm

Le paquet Glibmm est un ensemble de liaisons C++ pour Glib.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/glibmm/2.58/glibmm-2.58.0.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/glibmm/2.58/glibmm-2.58.0.tar.xz

  • Somme de contrôle MD5 du téléchargement : 4a0402ac65a7c47424b4e21b81b079c0

  • Taille du téléchargement : 6.5 Mo

  • Estimation de l'espace disque requis : 197 Mo (avec les tests)

  • Estimation du temps de construction : 1.2 SBU (avec parallelisme=4 ; avec les tests)

Dépendances de GLibmm

Requises

GLib-2.58.3 et libsigc++-2.10.1

Facultatives

Doxygen-1.8.15, glib-networking-2.58.0 (pour les tests), GnuTLS-3.6.6 (pour les tests) et libxslt-1.1.33

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/glibmm

Installation de GLibmm

Pour commencer, corrigez le nom du répertoire de documentation : 

sed -e '/^libdocdir =/ s/$(book_name)/glibmm-2.58.0/' \
    -i docs/Makefile.in

Installez GLibmm en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install
