Wayland-Protocols-1.17

Introduction à Wayland-Protocols

Le paquet Wayland-Protocols contient des protocoles Wayland supplémentaires qui ajoutent des fonctionnalités en dehors des protocoles déjà dans le cœur de Wayland.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Estimation de l'espace disque requis : 1.2 Mo (avec les tests)

Dépendances de Wayland-protocols

Requises

Wayland-1.16.0

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/wayland-protocols

Installation de Wayland-protocols

Installez Wayland-protocols en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install
