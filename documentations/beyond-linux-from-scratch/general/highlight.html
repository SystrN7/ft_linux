Highlight-3.49

Introduction à Highlight

Highlight est un utilitaire qui convertit le code source en text formaté avec de la coloration syntaxique.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://www.andre-simon.de/zip/highlight-3.49.tar.bz2

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : ea39fb9f562156efa35e4de586909d6a

  • Taille du téléchargement : 1.3 Mo

  • Estimation de l'espace disque requis : 27 Mo

  • Estimation du temps de construction : 0.3 SBU (avec parallélisme = 4)

Dépendances de Highlight

Requises

Boost-1.69.0 et Lua-5.3.5

Facultatives

Qt-5.12.1 (pour construire l'interface utilisateur)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/highlight

Installation de Highlight

Pour construire Highlight, lancez la commande suivante : 

make

Pour construire l'interface graphique Qt5, lancez la commande suivante : 

make gui

Ce paquet n'est pas fourni avec une suite de tests.

Pour installer Highlight, lancez la commande suivante en tant qu'utilisateur root : 

make install

Pour installer l'interface graphique, lancez la commande suivante en tant qu'utilisateur root : 

make install-gui
