DejaGnu-1.6.2

Introduction à DejaGnu

DejaGnu est un environnement de travail pour lancer des suites de test sur les outils GNU. Il est écrit en expect, ce qui utilise Tcl (Tool command language). Il a été installé par LFS dans le répertoire temporaire /tools. Ces instructions l'installent de façon permanente.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de DejaGnu

Installez DejaGnu en lançant les commandes suivantes : 

./configure --prefix=/usr &&
makeinfo --html --no-split -o doc/dejagnu.html doc/dejagnu.texi &&
makeinfo --plaintext       -o doc/dejagnu.txt  doc/dejagnu.texi

Pour tester l'installation, lancez make check.

Maintenant, en tant qu'utilisateur root : 

make install &&
install -v -dm755   /usr/share/doc/dejagnu-1.6.2 &&
install -v -m644    doc/dejagnu.{html,txt} \
                    /usr/share/doc/dejagnu-1.6.2

Descriptions courtes

runtest

est le programme d'exécution de tests DejaGnu. Il est utilisé pour contrôler quels tests seront lancés et avec quelles variantes sur leur exécution.
