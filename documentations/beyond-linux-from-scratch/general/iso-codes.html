ISO Codes-4.1

Introduction à ISO Codes

Le paquet ISO Codes contient une liste de noms de pays, de langues et de propriétés que l'on utilise en tant que base de données centrale pour l'accès à ces données.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/iso-codes

Installation de ISO Codes

Installez ISO Codes en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Pour tester les résultats lancez : make check.

Note

Si vous installez ISO codes sur une version précédemment installée, les étapes d'installation échouent à créer certains liens symboliques. Pour les mettre à jour correctement, lancez : 

sed -i '/^LN_S/s/s/sfvn/' */Makefile

Maintenant, en tant qu'utilisateur root : 

make install
