shared-mime-info-1.10

Introduction à Shared Mime Info

Le paquet shared-mime-info contient une base de données MIME. Ceci permet des mises à jour centrales d'informations MIME pour toutes les applications qui le supportent.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (FTP) :

Dépendances de Shared Mime Info

Requises

GLib-2.58.3 et libxml2-2.9.9

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/shared-mime-info

Installation de Shared Mime Info

Note

Si votre processeur n'est pas un Intel très récent, vous devrez utiliser make -j1 pour ce paquet.

Installez Shared Mime Info en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install
