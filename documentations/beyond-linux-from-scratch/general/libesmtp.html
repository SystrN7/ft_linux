libESMTP-1.0.6

Introduction à libESMTP

Le paquet libESMTP contient les bibliothèques libESMTP qui sont utilisées par certains programmes pour gérer l'envoi de messages électroniques à un transporteur de messages.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de libESMTP

Corrigez un problème de construction avec OpenSSL-1.1.0 : 

sed 's@SSL_library_init@SSL_new@g' -i configure.ac &&
autoreconf -fi

Installez libESMTP en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contenu

Programme installé: libesmtp-config
Bibliothèques installées: libesmtp.{so,a} et les greffons SASL de libESMTP
Répertoire installé: /usr/lib/esmtp-plugins

Descriptions courtes

libesmtp-config

affiche des informations de version et les options utilisées pour compiler libESMTP.

libesmtp.{so,a}

est utilisé pour gérer l'envoi de messages électroniques à un agent de transport du courrier.

greffons SASL de libesmtp

sont utilisées pour intégrer libesmtp à l'authentification SASL.
