Talloc-2.1.15

Introduction à Talloc

Talloc fournit un système de gestion de mémoire compté, hiérarchique et référentiel avec des destructeurs. C'est ce qui alloue de la mémoire et qui est utilisé dans Samba.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : https://www.samba.org/ftp/talloc/talloc-2.1.15.tar.gz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : a56bc6add28ae0f5693eb49e19ddddcf

  • Taille du téléchargement : 620 Ko

  • Estimation de l'espace disque requis : 10 Mo (avec les tests)

  • Estimation du temps de construction : 0.4 SBU (avec les tests)

Installation de Talloc

Installez Talloc en lançant les commandes suivantes : 

sed -i s:../../buildtools/bin/waf:buildtools/bin/waf: Makefile &&
./configure --prefix=/usr &&
make

Pour vérifier les résultats, lancez make check.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

sed -i s:[...] Makefile : Cette commande corrige un problème à la création de l'archive introduit avec cette version de Talloc.

--extra-python=PYTHON : Ce paramètre active la construction des bibliothèques sélectionnées pour les versions supplémentaires spécifiées de Python (exemple : --extra-python=/usr/bin/python2).
