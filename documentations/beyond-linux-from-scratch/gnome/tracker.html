Tracker-2.2.0

Introduction à Tracker

Tracker est le moteur de recherche et d'indexage utilisé dans l’environnement de bureau GNOME.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/tracker/2.2/tracker-2.2.0.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/tracker/2.2/tracker-2.2.0.tar.xz

  • Somme de contrôle MD5 du téléchargement : e4c637cbc1aa2f4d03e09c8e6b318d6d

  • Taille du téléchargement : 2.7 Mo

  • Estimation de l'espace disque requis : 211 Mo (avec les tests)

  • Estimation du temps de construction : 1.0 SBU (avec les tests)

Dépendances de Tracker

Requises

JSON-GLib-1.4.4, libseccomp-2.3.3, libsoup-2.64.2 et Vala-0.42.5

Recommandées

gobject-introspection-1.58.3, ICU-63.1, NetworkManager-1.14.6, SQLite-3.27.1 et UPower-0.99.9

Facultatives

GTK-Doc-1.29, Python-2.7.15 (pour les tests « fonctionnels ») et Snowball

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/tracker

Installation de Tracker

Installez Tracker en lançant les commandes suivantes : 

mkdir build &&
cd build &&

meson --prefix=/usr --sysconfdir=/etc .. &&
ninja

Pour tester les résultats, lancez : ninja test. Les tests devraient être lancés depuis une session graphique. Plusieurs tests peuvent afficher une erreur si le module Python 2 de PyGObject3 n'est pas disponible.

Maintenant, en tant qu'utilisateur root : 

ninja install

Contents

Programmes installés: tracker
Bibliothèque installée: libtracker-control.so, libtracker-miner.so et libtracker-sparql.so
Répertoires installés: /usr/{include,lib}/tracker-2.0, /usr/share/tracker et /usr/share/gtk-doc/libtracker-{control,miner,sparql}

Descriptions courtes

tracker

est le programme de contrôle de l'indexeur

libtracker-control.so

contient les fonctions de contrôle et de surveillance des mineurs.

libtracker-miner.so

contient les fonctions de fouille de données.

libtracker-sparql.so

contient les fonctions de gestion de ressources et de bases de données.
