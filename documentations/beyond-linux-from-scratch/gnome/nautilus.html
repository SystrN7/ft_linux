Nautilus-3.30.5

Introduction à Nautilus

Le paquet Nautilus contient le gestionnaire de fichiers de GNOME.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/nautilus/3.30/nautilus-3.30.5.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/nautilus/3.30/nautilus-3.30.5.tar.xz

  • Somme de contrôle MD5 du téléchargement : e9d392e7a11440d4d981ce49d8960d5b

  • Taille du téléchargement : 3.1 Mo

  • Estimation de l'espace disque requis : 147 Mo (avec les tests)

Installation de Nautilus

Installez Nautilus en lançant les commandes suivantes : 

sed s/\'libm\'/\'m\'/ -i meson.build &&
mkdir build &&
cd    build &&

meson --prefix=/usr      \
      --sysconfdir=/etc  \
      -Dselinux=false    \
      -Dpackagekit=false \
      .. &&

ninja

Pour tester les résultats, lancez : ninja test. Les tests doivent être lancés dans un environnement graphique.

Maintenant, en tant qu'utilisateur root : 

ninja install &&
glib-compile-schemas /usr/share/glib-2.0/schemas

Explication des commandes

sed s/\'libm\'/\'m\'/ -i meson.build : Corrigez le système de construction pour qu'il trouve la bibliothèque Math.

-Dselinux=false : Ce paramètre désactive l'utilisation de selinux qui n'est pas supporté par BLFS.

-Dpackagekit=false : Ce paramètre désactive l'utilisation de PackageKit qui n'est pas utilisable dans BLFS.

Contenu

Programmes installés: nautilus et nautilus-autorun-software
Bibliothèque installée: libnautilus-extension.so
Répertoires installés: /usr/{include,lib}/nautilus et /usr/share/{gnome-shell/search-providers,gtk-doc/html/libnautilus-extension}

Descriptions courtes

nautilus

est le gestionnaire de fichiers de GNOME.

libnautilus-extension.so

fournit les fonctions requises par les extensions du gestionnaire de fichiers.
