Cheese-3.30.0

Introduction à Cheese

Cheese est utilisé pour prendre des photos et des films avec votre webcam, avec des effets graphiques rigolos.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Configuration du noyeau

En fonction de votre webcam, activez les options appropriées et recompilez le noyau si nécessaire : 

Device Drivers  --->
  Multimedia support --->
    <*> Cameras/video grabbers support  [CONFIG_MEDIA_CAMERA_SUPPORT]
    <*> Media USB Adapters  --->         [CONFIG_MEDIA_USB_SUPPORT]
              Select device(s) as needed

Installation de Cheese

Tout d'abord corrigez un avertissement d'obsolescence qui peut rapidement remplir le journal : 

sed -i '7d' data/cheese.css

Installez Cheese en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'a pas de suite de tests valide.

Maintenant, en tant qu'utilisateur root : 

make install
