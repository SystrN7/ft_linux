gnome-nettool-3.8.1

Introduction à GNOME Nettool

Le paquet gnome-nettool est un outil d'information réseau qui donne aux utilisateurs des interfaces GUI avec certains des outils en ligne de commande les plus fréquents.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/gnome-nettool/3.8/gnome-nettool-3.8.1.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/gnome-nettool/3.8/gnome-nettool-3.8.1.tar.xz

  • Somme de contrôle MD5 du téléchargement : d4fc944b7ba8fd5f49f04a73e0d37e80

  • Estimation du temps de construction : 0.1 SBU

Dépendances de GNOME Nettool

Requises

GTK+-3.24.5, itstool-2.0.5 et libgtop-2.38.0

Dépendances de Runtime

BIND Utilities-9.12.3-P4, Nmap-7.70, Net-tools-CVS_20101030, Traceroute-2.1.0 et Whois-5.4.1

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/gnome-nettool

Installation de GNOME Nettool

Installez GNOME Nettool en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
