Introduction à GNOME Calculator

GNOME Calculator est une puissante calculatrice graphique avec des modes financier, logique et scientifique. Elle utilise un paquet à précision multiple pour faire ses calculs pour donner une grande précision.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/gnome-calculator

Installation de GNOME Calculator

Installez GNOME Calculator en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr .. &&
ninja

Pour tester les résultats lancez : ninja test.

Maintenant, en tant qu'utilisateur root : 

ninja install

Contenu

Bibliothèque installée: None
Répertoires installés: /usr/share/help/*/gnome-calculator

Descriptions courtes

gnome-calculator

gcalccmd
