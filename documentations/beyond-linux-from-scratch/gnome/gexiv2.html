gexiv2-0.11.0

Introduction à gexiv2

gexiv2 est une enveloppe basée sur GObject autour de la bibliothèque Exiv2.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : https://download.gnome.org/sources/gexiv2/0.11/gexiv2-0.11.0.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/GNOME/sources/gexiv2/0.11/gexiv2-0.11.0.tar.xz

  • Somme de contrôle MD5 du téléchargement : 6b9d0d9901df847a49903f11f3d6f685

  • Taille du téléchargement : 364 Ko

  • Estimation de l'espace disque requis : 5.1 Mo

  • Estimation du temps de construction : 0.1 SBU

Dépendances de gexiv2

Requises

Exiv2-0.27.0

Recommandées

Vala-0.42.5

Facultatives

GTK-Doc-1.29 (pour la documentation) et Python-2.7.15 (pour le module Python2)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/gexiv2

Installation de gexiv2

Installez gexiv2 en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr .. &&
ninja

Ce paquet n'est pas fourni avec une suite de tests.

En tant qu'utilisateur root : 

ninja install
