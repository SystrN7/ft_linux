Seahorse-3.30.1.1

Introduction à Seahorse

Seahorse est une interface graphique pour gérer et utiliser des clés de chiffrement. Il supporte actuellement les clés PGP (en utilisant GPG/GPGME) et les clés SSH.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de Seahorse

Installez Seahorse en lançant les commandes suivantes : 

sed -i -r 's:"(/apps):"/org/gnome\1:' data/*.xml &&

mkdir build &&
cd build &&

meson --prefix=/usr .. &&
ninja

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

ninja install

Explication des commandes

sed ... data/*.xml : Cette commande corrige des entrées obsolètes dans les modèles de schéma.
