network-manager-applet-1.8.20

Introduction à NetworkManager Applet

NetworkManager Applet fournit un outil et un applet utilisé pour configurer les connexions filaires et sans fils depuis une interface graphique. Il est conçu pour être utilisé avec n'importe quel environnement de bureau utilisant GTK+ comme Xfce et LXDE.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de NetworkManager Applet

Installez NetworkManager Applet en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr     \
      --sysconfdir=/etc \
      -Dselinux=false   \
      -Dteam=false      \
      -Dmobile_broadband_provider_info=false .. &&
ninja

Pour tester les résultats lancez : ninja test.

Maintenant, en tant qu'utilisateur root : 

ninja install

Explication des commandes

-Dteam=false : Ce paramètre désactive l'éditeur de configuration team car il requiert jansson-2.12. Si vous avez installé Jansson, supprimez cette option ou mettez-la à « true ».

-Denable-selinux=false : Ce paramètre force la désactivation du support pour SELinux car il n'est actuellement pas dans BLFS et la construction échoue sans lui.

-Dmobile_broadband_provider_info=false : Ce paramètre force la désactivation du support pour Mobile Broadband Provider qui n'est actuellement pas disponible dans BLFS.

-Dwwan=false : Ce paramètre désactive le support de WWAN. Utilisez-le si vous n'avez pas installé ModemManager-1.10.0.

-Dintrospection=false : Utilisez cela si vous n'avez pas installé gobject-introspection-1.58.3.

-Dgtk_doc=false : Utilisez cela si vous n'avez pas installé GTK-Doc-1.29.
