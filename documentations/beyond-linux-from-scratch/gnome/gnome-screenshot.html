gnome-screenshot-3.30.0

Introduction à GNOME Screenshot

GNOME Screenshot est un utilitaire utilisé pour prendre des captures d'écran de l'écran entier, d'une fenêtre ou une zone définie par l'utilisateur, avec en option des effets de bordure enjolivante.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de GNOME Screenshot

Note

Il y a un problème de construction causé par un fichier installé par gettext dans LFS. Si gettext a été installé avec les instructions d'avant le 11 avril 2018, lancez ce qui suit en tant qu'utilisateur root : 

sed -e '/AppData/N;N;p;s/\.appdata\./.metainfo./' \
    -i /usr/share/gettext-0.19.8/its/appdata.loc

Installez GNOME Screenshot en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr .. &&
ninja

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

ninja install

Note

Afin de mieux exécuter GNOME Screenshot depuis la ligne de commandes, l'option --interactive doit être spécifiée.
