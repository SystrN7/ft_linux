yelp-xsl-3.30.1

Introduction à Yelp XSL

Le paquet yelp-xsl contient les feuilles de style XSL qui sont utilisées par le navigateur d'aide Yelp pour formater les documents Docbook et Mallard.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/yelp-xsl/3.30/yelp-xsl-3.30.1.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/yelp-xsl/3.30/yelp-xsl-3.30.1.tar.xz

  • Somme de contrôle MD5 du téléchargement : 371f7379f1614e602f861922182bbfbe

  • Taille du téléchargement : 508 Ko

  • Estimation de l'espace disque requis : 8.5 Mo

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de Yelp XSL

Requises

libxslt-1.1.33 et itstool-2.0.5

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/yelp-xsl

Installation de Yelp XSL

Installez Yelp XSL en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
