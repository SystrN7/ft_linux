VTE-0.54.3

Introduction à VTE

Le paquet VTE contient une implémentation du fichier termcap pour les émulateurs de terminal.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/vte/0.54/vte-0.54.3.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/vte/0.54/vte-0.54.3.tar.xz

  • Somme de contrôle MD5 du téléchargement : fc0e9de2b6199c69e529ff3629fa35c8

  • Taille du téléchargement : 1.0 Mo

Dépendances de VTE

Requises

GTK+-3.24.5, libxml2-2.9.9 et pcre2-10.32

Recommandées

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/vte

Installation de VTE

Installez VTE en lançant les commandes suivantes : 

./configure --prefix=/usr          \
            --sysconfdir=/etc      \
            --disable-static       \
            --enable-introspection &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--enable-introspection : Ce paramètre active les liaisons de Gobject Introspection. Effacez-le si vous n'avez pas installé gobject-introspection-1.58.3.

--without-gnutls : Ajoutez ce paramètre si vous n'avez pas installé GnuTLS-3.6.6.

--disable-static : Ce paramètre empêche l'installation des versions statiques des bibliothèques.

--enable-gtk-doc : Utilisez ce paramètre si GTK-Doc est installé et si vous souhaitez reconstruire et installer la documentation de l'API.

libvte-2.91.so
