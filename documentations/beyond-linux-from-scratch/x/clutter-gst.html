Introduction à Clutter Gst

Le paquet Clutter Gst contient une bibliothèque d'intégration pour utiliser GStreamer avec Clutter. Son but est d'implémenter l'interface ClutterMedia en utilisant GStreamer.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de Clutter Gst

Installez Clutter Gst en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'a pas de suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
