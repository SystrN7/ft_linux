Introduction à Breeze Icons

Le paquet Breeze Icons contient un thème d'icônes pour les applications KDE Plasma 5, mais peut être utilisé pour d'autres environnements fenêtrés.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Dépendances de Breeze Icons

Requises

Facultatives

KDE Frameworks-5.55.0

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/breeze-icons

Installation de Breeze Icons

Installez Breeze Icons en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DBUILD_TESTING=OFF         \
      -Wno-dev ..

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DBUILD_TESTING=OFF : Ce paramètre désactive de la construction les outils de tests, qui nécessitent quelques modules de KDE Frameworks-5.55.0.
