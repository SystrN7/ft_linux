Atkmm-2.28.0

Introduction à Atkmm

Atkmm est l'interface officielle en C++ de la bibliothèque d'accessoires pour l'accessibilité ATK.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/atkmm/2.28/atkmm-2.28.0.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/atkmm/2.28/atkmm-2.28.0.tar.xz

  • Somme de contrôle MD5 du téléchargement : 6194ac577f15567adfa3c923944c6651

  • Taille du téléchargement : 872 Ko

  • Estimation de l'espace disque requis : 16 Mo

  • Estimation du temps de construction : 0.2 SBU

Dépendances de Atkmm

Requises

ATK-2.30.0 et GLibmm-2.58.0

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/atkmm

Installation de Atkmm

Pour commencer, corrigez le nom du répertoire de documentation : 

sed -e '/^libdocdir =/ s/$(book_name)/atkmm-2.28.0/' \
    -i doc/Makefile.in

Installez Atkmm en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'a pas de suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
