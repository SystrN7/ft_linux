Pangomm-2.42.0

Introduction à Pangomm

Le paquet pangomm fournit une interface C++ pour Pango.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de Pangomm

Pour commencer, corrigez le nom du répertoire de documentation : 

sed -e '/^libdocdir =/ s/$(book_name)/pangomm-2.42.0/' \
    -i docs/Makefile.in

Installez Pangomm en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
