Cairomm-1.12.2

Introduction à Cairomm

Le paquet Cairomm fourni une interface C++ pour Cairo.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : https://www.cairographics.org/releases/cairomm-1.12.2.tar.gz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 9d2282ea34cf9aaa89208bb4bb911909

  • Taille du téléchargement : 1.3 Mo

  • Estimation de l'espace disque requis : 11 Mo

  • Estimation du temps de construction : 0.1 SBU

Dépendances de Cairomm

Requises

Cairo-1.16.0 et libsigc++-2.10.1

Facultatives

Boost-1.69.0 et Doxygen-1.8.15

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/cairomm

Installation de Cairomm

Pour commencer, corrigez le nom du répertoire de documentation : 

sed -e '/^libdocdir =/ s/$(book_name)/cairomm-1.12.2/' \
    -i docs/Makefile.in

Installez Cairomm en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
