Introduction à GNOME Themes Standard

Le paquet GNOME Themes Standard contient divers composants du thème GNOME par défaut.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.3.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/gnome-themes-standard/3.22/gnome-themes-standard-3.22.3.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/gnome-themes-standard/3.22/gnome-themes-standard-3.22.3.tar.xz

  • Somme de contrôle MD5 du téléchargement : b51c362b157b6407303d44f93c31ee11

  • Taille du téléchargement : 2.8 Mo

  • Estimation de l'espace disque requis : 40 Mo

  • Estimation du temps de construction : 0.3 SBU

Dépendances de GNOME Themes Standard

Requises

GTK+-2.24.32 ou GTK+-3.24.0 avec librsvg-2.42.2 ou les deux

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/gnome-themes-standard

Installation de GNOME Themes Standard

Installez GNOME Themes Standard en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--disable-gtk2-engine : Cette option désactive le moteur de thème GTK+-2.

--disable-gtk3-engine : Cette option désactive le moteur de thèmes GTK+-3.

Contenu

Programmes installés: Aucun
Bibliothèque installée: libadwaita.so
Répertoires installés: /usr/lib/gtk-2.0/2.10.0/engines, /usr/share/icons/HighContrast, /usr/share/themes/Adwaita, /usr/share/themes/Adwaita-dark et /usr/share/themes/HighContrast
