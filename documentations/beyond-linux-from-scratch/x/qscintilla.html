QScintilla-gpl-2.10.4

Introduction à QScintilla

QScintilla est un port de Scintilla sur Qt. En plus des fonctions trouvées dans les composants standards d'édition de texte, il ajoute des fonctions particulièrement utiles pour l'édition et le débogage de code source : la coloration syntaxique, l'indication d'erreurs, la complétion de code, un assistant fonction, le pliage de code, des marqueurs dans les marges comme dans les débogueurs pour indiquer les points d'arrêts et la ligne courante, les macros enregistrables, les vues multiples et, bien sur, l'impression.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Installation de QScintilla

Note

Les instructions dans cette page sont juste pour l'application QScintilla. Le code est inclus dans l'archive pour le greffon QScintilla Qt Designer et les liaisons Python.

Installez QScintilla en lançant les commandes suivantes : 

cd Qt4Qt5             &&
qmake qscintilla.pro  &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install &&
ln -sfv $(readlink $QT5DIR/lib/libqscintilla2_qt5.so) $QT5DIR/lib/libqt5scintilla2.so

Si vous souhaitez installer la documentation, tapez, en tant qu'utilisateur root : 

install -v -m755 -d $QT5DIR/share/doc/QScintilla-2.10.4/html &&
install -v -m644    ../doc/html-Qt4Qt5/* \
                    $QT5DIR/share/doc/QScintilla-2.10.4/html

Explication des commandes

ln -sfv ... : Cette commande crée un lien symbolique de compatibilité utilisé par certains programmes.
