libepoxy-1.5.3

Introduction à libepoxy

libepoxy est une bibliothèque pour prendre en charge la gestion des pointeurs de fonction OpenGL.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de libepoxy

Installez libepoxy en lançant les commandes suivantes : 

mkdir build &&
cd build &&

meson --prefix=/usr .. &&
ninja

Pour tester les résultats lancez : ninja test.

Maintenant, en tant qu'utilisateur root : 

ninja install
