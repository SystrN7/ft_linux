Introduction à Adwaita Icon Theme

Le paquet Adwaita Icon Theme contient un thème d'icônes pour les applications Gtk+ 3.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/adwaita-icon-theme/3.30/adwaita-icon-theme-3.30.1.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/adwaita-icon-theme/3.30/adwaita-icon-theme-3.30.1.tar.xz

  • Somme de contrôle MD5 du téléchargement : db3fd812821d72fdd9a3c7d622a95b35

  • Taille du téléchargement : 19 Mo

  • Estimation de l'espace disque requis : 135 Mo

  • Estimation du temps de construction : 0.9 SBU

Dépendances de Adwaita Icon Theme

Facultatives

git-2.20.1, GTK+-2.24.32 ou GTK+-3.24.5 (si présent, librsvg-2.44.12 est aussi requis, et gtk-update-icon-cache et gtk-encode-symbolic-svg sont lancés après l'installation), Inkscape-0.92.4 et Icon Tools

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/adwaita-icon-theme

Installation de Adwaita Icon Theme

Installez Adwaita Icon Theme en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contenu

Programmes installés: Aucun
Bibliothèque installée: Aucune
Répertoire installé: /usr/share/icons/Adwaita
