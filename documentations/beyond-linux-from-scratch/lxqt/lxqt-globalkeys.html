lxqt-globalkeys-0.12.0

Introduction à lxqt-globalkeys

Le paquet lxqt-globalkeys contient un démon utilisé pour enregistrer les raccourcis clavier globaux et un éditeur graphique de raccourcis.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Taille du téléchargement : 51 Ko

  • Estimation de l'espace disque requis : 9.3 Mo

Dépendances de lxqt-globalkeys

Requises

liblxqt-0.12.0

Facultatives

git-2.17.1 (pour télécharger les fichiers de traductions) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisations)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-globalkeys

Installation de lxqt-globalkeys

Installez lxqt-globalkeys en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.

Contenu

Programmes installés: lxqt-config-globalkeyshortcuts et lxqt-globalkeysd
Bibliothèques installées: liblxqt-globalkeys.so et liblxqt-globalkeys-ui.so
Répertoires installés: $LXQT_PREFIX/include/lxqt-globalkeys{,-ui} et $LXQT_PREFIX/share/cmake/lxqt-globalkeys{,-ui}
