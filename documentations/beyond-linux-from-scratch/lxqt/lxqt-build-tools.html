lxqt-build-tools-0.4.0

Introduction à lxqt-build-tools

Le paquet lxqt-build-tools fournit divers outils requis pour construire LXQt ainsi que d'autres composants maintenus par le projet LXQt.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (FTP) :

  • Taille du téléchargement : 20 Ko

  • Estimation de l'espace disque requis : 780 Ko

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de lxqt-build-tools

Requises

CMake-3.12.0 et Qt-5.11.1

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-build-tools

Installation de lxqt-build-tools

Installez lxqt-build-tools en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DCMAKE_BUILD_TYPE=Release  \
      .. &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
