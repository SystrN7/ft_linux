qterminal-0.8.0

Introduction à qterminal

Le paquet qterminal contient un émulateur de terminal basé sur les widgets de Qt pour qt avec le support des onglets multiples.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/qterminal/releases/download/0.8.0/qterminal-0.8.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 1ce5911e0959da75e6742f10a36b73e4

  • Taille du téléchargement : 102 Ko

  • Estimation de l'espace disque requis : 6.3 Mo (plus 1.3 Mo pour la documentation de l'API)

  • Estimation du temps de construction : 0.3 SBU (avec la documentation de l'API)

Dépendances de qterminal

Requises

lxqt-build-tools-0.4.0 et qtermwidget-0.8.0

Facultatives

Doxygen-1.8.14 et texlive-20180414 (ou install-tl-unx), pour la documentation et git-2.17.1 (pour télécharger les fichiers de traduction) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisation)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/qterminal

Installation de qterminal

Installez qterminal en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DCMAKE_BUILD_TYPE=Release          \
      -DPULL_TRANSLATIONS=no              \
      ..       &&

make

Si vous avez installé Doxygen-1.8.14, et souhaitez créer la documentation de l'API, lancez : 

doxygen ../Doxyfile

La documentation de l'API en HTML a été généré avec la commande précédente. Maintenant, si vous avez aussi installé texlive-20180414 (ou install-tl-unx) et souhaitez créer le manuel de référence de l'API en pdf, lancez : 

make -C docs/latex

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Pour installer la documentation de l'API, si vous avez décidé de la générer, lancez en tant qu'utilisateur root : 

install -v -m755 -d /usr/share/doc/qterminal-0.8.0/{html,pdf} &&
cp -vr docs/html/* /usr/share/doc/qterminal-0.8.0/html

Le manuel de référence de l'API en pdf peut être installé avec, en tant qu'utilisateur root : 

install -v -m644    docs/latex/refman.pdf \
                    /usr/share/doc/qterminal-0.8.0/pdf

Si vous installez dans /opt, de nouveau en tant qu'utilisateur root : 

if [ "$LXQT_PREFIX" != /usr ]; then
  ln -svf $LXQT_PREFIX/share/applications/qterminal.desktop \
          /usr/share/applications
  ln -svf $LXQT_PREFIX/share/applications/qterminal_drop.desktop \
          /usr/share/applications
fi

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.
