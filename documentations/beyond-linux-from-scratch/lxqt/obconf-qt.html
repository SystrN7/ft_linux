obconf-qt-0.12.0

Introduction à obconf-qt

Le paquet obconf-qt est un outil de configuration OpenBox basé sur Qt.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/obconf-qt/releases/download/0.12.0/obconf-qt-0.12.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 91e6e57e80380437b9db2622745dfd4e

  • Taille du téléchargement : 33 Ko

  • Estimation de l'espace disque requis : 3.0 Mo

  • Estimation du temps de construction : 0.2 SBU

Dépendances de obconf-qt

Requises

GTK+-2.24.32 (pour gtk-update-icon-cache), hicolor-icon-theme-0.17, liblxqt-0.12.0 et openbox-3.6.1

Facultatives

git-2.17.1 (pour télécharger les fichiers de traduction) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisation)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/obconf-qt

Installation de obconf-qt

Installez obconf-qt en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Si vous installez dans /opt, de nouveau en tant qu'utilisateur root : 

if [ "$LXQT_PREFIX" != /usr ]; then
  ln -svf $LXQT_PREFIX/share/applications/obconf-qt.desktop \
          /usr/share/applications
fi

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.
