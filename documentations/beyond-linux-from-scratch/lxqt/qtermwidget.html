qtermwidget-0.8.0

Introduction à qtermwidget

Comme le nom le suggère, qtermwidget est un widget de terminal pour Qt.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/qtermwidget/releases/download/0.8.0/qtermwidget-0.8.0.tar.xz

  • Taille du téléchargement : 157 Ko

  • Estimation de l'espace disque requis : 7.7 Mo

  • Estimation du temps de construction : 0.3 SBU

Dépendances de qtermwidget

Requises

lxqt-build-tools-0.4.0

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/qtermwidget

Installation de qtermwidget

Installez qtermwidget en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DBUILD_TEST=true : Construit l'application de test.
