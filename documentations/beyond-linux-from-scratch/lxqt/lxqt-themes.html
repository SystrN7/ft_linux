lxqt-themes-0.12.0

Introduction à lxqt-themes

Le paquet lxqt-themes fournit un certain nombre de fichiers graphiques et de thèmes pour le bureau LXQt.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/lxqt-themes/releases/download/0.12.0/lxqt-themes-0.12.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : e468462aa42b6d8e6e7d24403e100226

  • Taille du téléchargement : 3.1 Mo

  • Estimation de l'espace disque requis : 8.2 Mo

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de lxqt-themes

Requises

lxqt-build-tools-0.4.0,

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-themes

Installation de lxqt-themes

Installez lxqt-themes en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

mkdir -pv $LXQT_PREFIX/share/lxqt/graphics &&
make install
