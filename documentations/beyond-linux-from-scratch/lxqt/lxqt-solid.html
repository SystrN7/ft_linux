Solid-5.46.0 pour lxqt

Introduction à solid

Solid est une boîte à outils d'intégration pour les périphériques. Il fournit une manière d'envoyer des requêtes et d'interagir avec le matériel indépendamment du système d'exploitation sous-jacent.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Note

Ce paquet est extrait de l'ensemble de paquet de KF5. Si KDE Frameworks-5.46.0 est installé, ne construisez PAS ce paquet aussi comme présenté ici.

Installation de solid

Installez solid en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DCMAKE_BUILD_TYPE=Release          \
      -DBUILD_TESTING=OFF                 \
      -Wno-dev ..                         &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
