lxqt-notificationd-0.12.0

Introduction à lxqt-notificationd

Le paquet lxqt-notificationd est le démon de notification de LXQT.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/lxqt-notificationd/releases/download/0.12.0/lxqt-notificationd-0.12.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 1cf3e6aaeb2cc4b3973d493ad7fd3503

  • Taille du téléchargement : 31 Ko

  • Estimation de l'espace disque requis : 3.7 Mo

  • Estimation du temps de construction : 0.2 SBU

Dépendances de lxqt-notificationd

Requises

liblxqt-0.12.0

Facultatives

git-2.17.1 (pour télécharger les fichiers de traduction) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisation)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-notificationd

Installation de lxqt-notificationd

Installez lxqt-notificationd en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.
