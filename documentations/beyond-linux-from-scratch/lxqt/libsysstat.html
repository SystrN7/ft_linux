libsysstat-0.4.0

Introduction à libsysstat

Le paquet libsysstat contient une bibliothèque utilisée pour demander des informations et des statistiques du système.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/libsysstat/releases/download/0.4.0/libsysstat-0.4.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 63fbaf270a259dcdf4a9fefa91f5e88d

  • Taille du téléchargement : 17 Ko

  • Estimation de l'espace disque requis : 1.6 Mo

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de libsysstat

Requises

CMake-3.12.0, lxqt-build-tools-0.4.0 et Qt-5.11.1

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/libsysstat

Installation de libsysstat

Installez libsysstat en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Contenu

Programme installé: Aucun
Bibliothèque installée: libsysstat-qt5.so
Répertoires installés: $LXQT_PREFIX/include/sysstat-qt5 et $LXQT_PREFIX/share/cmake/sysstat-qt5
