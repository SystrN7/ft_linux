Kwindowsystem-5.46.0 pour lxqt

Introduction à kwindowsystem

Le paquet kwindowsystem fournit des informations à propos du système de fenêtrage et permet d'interagir avec. Il fournit une API de haut niveau indépendante du système de fenêtrage et a des implémentations spécifiques à la plateforme.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Note

Ce paquet est extrait de l'ensemble de paquets KF5. Si KDE Frameworks-5.46.0 est construit, ne construisez PAS ce paquet aussi comme présenté ici.

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/kwindowsystem

Installation de kwindowsystem

Installez kwindowsystem en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DCMAKE_BUILD_TYPE=Release          \
      -DBUILD_TESTING=OFF                 \
      -Wno-dev ..                         &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
