Libkscreen-5.12.5 pour lxqt

Introduction à libkscreen

Le paquet Libkscreen contient la bibliothèque de gestion de l'écran de KDE.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Note

Ce paquet est extrait de l'ensemble de paquets KDE Plasma. Si Plasma-5.12.5 est construit, ne construisez PAS ce paquet comme présenté ici.

Informations sur le paquet

  • Téléchargement (HTTP) : http://download.kde.org/stable/plasma/5.12.5/libkscreen-5.12.5.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 1e3b3aa0026bf51121fa36f4380db6e5

  • Taille du téléchargement : 96 Ko

  • Estimation de l'espace disque requis : 9.6 Mo

  • Estimation du temps de construction : 0.2 SBU (avec parallélisme = 4)

Dépendances de Libkscreen

Requises

kwayland-5.46.0 for lxqt

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/kwayland

Installation de libkscreen

Installez libkscreen en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_LIBDIR=lib          \
      -DBUILD_TESTING=OFF                 \
      -Wno-dev ..                         &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
