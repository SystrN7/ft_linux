lximage-qt-0.6.0

Introduction à lximage-qt

Le paquet lximage-qt contient une visionneuse d'images légère et un programme de capture d'écran.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Installation de lximage-qt

Installez lximage-qt en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Si vous avez installé dans /opt, de nouveau en tant qu'utilisateur root : 

ln -svf $LXQT_PREFIX/share/applications/lximage-qt.desktop \
        /usr/share/applications &&

ln -svf $LXQT_PREFIX/share/applications/lximage-qt-screenshot.desktop \
        /usr/share/applications

Si vous avez installé xdg-utils-1.1.3, en tant qu'utilisateur root, lancez xdg-icon-resource forceupdate --theme hicolor.

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.

Descriptions courtes

lximage-qt

est une visionneuse d'images légère et un programme de capture d'écran. Il y a une entrée de menu dans Utilitaires pour le programme de capture d'écran. Depuis la ligne de commande, vous pouvez obtenir une capture de tout l'écran ou d'une fenêtre en lançant la commande lximage-qt --screenshot.
