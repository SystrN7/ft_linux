lxqt-policykit-0.12.0

Introduction à lxqt-policykit

Le paquet lxqt-policykit est l'agent PolicyKit de LXQT.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/lxqt-policykit/releases/download/0.12.0/lxqt-policykit-0.12.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 8d0380fd7d629e5416bad60caed16980

  • Taille du téléchargement : 32 Ko

  • Estimation de l'espace disque requis : 1.5 Mo

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de lxqt-policykit

Requises

liblxqt-0.12.0, polkit-qt-0.112.0

Facultatives

git-2.17.1 (pour télécharger les fichiers de traduction) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisation)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-policykit

Installation de lxqt-policykit

Installez lxqt-policykit en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DCMAKE_BUILD_TYPE=Release          \
      -DPULL_TRANSLATIONS=no              \
      ..                                  &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.
