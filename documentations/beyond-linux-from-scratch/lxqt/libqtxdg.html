libqtxdg-3.1.0

Introduction à libqtxdg

La bibliothèque libqtxdg est une implémentation en Qt des spécifications xdg de freedesktop.org.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/libqtxdg/releases/download/3.1.0/libqtxdg-3.1.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 15db87c252dfdf6a6852abb08a749965

  • Taille du téléchargement : 65 Ko

  • Estimation de l'espace disque requis : 5.1 Mo

  • Estimation du temps de construction : 0.2 SBU

Dépendances de libqtxdg

Requises

CMake-3.12.0 et Qt-5.11.1

Facultatif (exécution)

GTK+-2.24.32 ou GTK+-3.22.30 (pour gtk-update-icon-cache)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/libqtxdg

Installation de libqtxdg

Installez libqtxdg en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DBUILD_TESTS=ON : Construit les tests.

-DBUILD_DEV_UTILS=ON : Construire et installer les utilitaires de développement.

Contenu

Programmes installés: Aucun
Bibliothèque installée: libQt5Xdg.so et libQt5XdgIconLoader.so
Répertoires installés: $LXQT_PREFIX/include/{qt5xdg,qt5xdgiconloader} et $LXQT_PREFIX/share/cmake/{qt5xdg,qt5xdgiconloader}
