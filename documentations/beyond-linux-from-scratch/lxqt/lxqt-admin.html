lxqt-admin-0.12.0

Introduction à lxqt-admin

Le paquet lxqt-admin fournit deux outils en interfaces graphiques pour ajuster les options du système d'exploitation sur lequel tourne LXQT.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (FTP) :

  • Taille du téléchargement : 113 Ko

  • Estimation de l'espace disque requis : 3.5 Mo

  • Estimation du temps de construction : 0.2 SBU

Dépendances de lxqt-admin

Requises

liblxqt-0.12.0 et Polkit-0.114

Facultatives

git-2.17.1 (pour télécharger les fichiers de traductions) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisations)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-admin

Installation de lxqt-admin

Installez lxqt-admin en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.

Contents

Descriptions courtes

lxqt-admin-time

est une interface graphique qui permet d'ajuster la date et l'heure.

lxqt-admin-user

est une interface graphique pour gérer les groupes et les utilisateurs.
