lxqt-l10n-0.12.0

Introduction à lxqt-l10n

Le paquet lxqt-l10n fournit des traductions pour tous les composants du bureau LXQT.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/lxqt-l10n/releases/download/0.12.0/lxqt-l10n-0.12.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 7f562caa4ad504f80f15985f1d48290d

  • Taille du téléchargement : 679 Ko

  • Estimation de l'espace disque requis : 37 Mo pour l'ensemble complet des traductions

  • Estimation du temps de construction : 0.1 SBU

Dépendances de lxqt-l10n

Requises

liblxqt-0.12.0

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-l10n

Installation de lxqt-l10n

Installez lxqt-l10n en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DWITH_<PACKAGE>=no : Ne pas installer de traductions pour PACKAGE, où PACKAGE est COMPTON_CONF, LIBFM_QT, LIBLXQT, LXIMAGE_QT, LXQT_ABOUT, LXQT_ADMIN, LXQT_CONFIG, LXQT_GLOBALKEYS, LXQT_NOTIFICATIOND, LXQT_OPENSSH_ASKPASS, LXQT_PANEL, LXQT_POLICYKIT, LXQT_POWERMANAGEMENT, LXQT_RUNNER, LXQT_SESSION, LXQT_SUDO, OBCONF_QT, PCMANFM_QT ou QTERMINAL.
