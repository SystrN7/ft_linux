pavucontrol-qt-0.3.0

Introduction à pavucontrol-qt

pavucontrol-qt est le port Qt du contrôleur de volume pavucontrol du serveur de son PulseAudio. Son utilisation n'est pas limitée à LXQt.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Dépendances de pavucontrol-qt

Requises

liblxqt-0.12.0 et PulseAudio-11.1 construit avec le support de GLib-2.56.1

Facultatives

git-2.17.1 (pour télécharger les fichiers de traduction) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisation)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/pavucontrol-qt

Installation de pavucontrol-qt

Installez pavucontrol-qt en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Si vous installez dans /opt, de nouveau en tant qu'utilisateur root : 

if [ "$LXQT_PREFIX" != /usr ]; then
  ln -svf $LXQT_PREFIX/share/applications/pavucontrol-qt.desktop \
          /usr/share/applications
fi

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.
