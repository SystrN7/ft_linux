lxqt-about-0.12.0

Introduction à lxqt-about

Le paquet lxqt-about fournit la boîte de dialogue indépendante « À propos » de LXQT.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/lxqt-about/releases/download/0.12.0/lxqt-about-0.12.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 92df8e8f8394b24cf9823009835afb00

  • Taille du téléchargement : 22 Ko

  • Estimation de l'espace disque requis : 1.8 Mo

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de lxqt-about

Requises

liblxqt-0.12.0

Facultatives

git-2.17.1 (pour télécharger les fichiers de traductions) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisations)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-about

Installation de lxqt-about

Installez lxqt-about en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.
