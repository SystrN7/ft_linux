lxqt-panel-0.12.0

Introduction à lxqt-panel

Le paquet lxqt-panel contient un panneau de bureau X11 léger.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/lxqt-panel/releases/download/0.12.0/lxqt-panel-0.12.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 58bf215ddfb9727d8c494618c58064e5

  • Taille du téléchargement : 265 Ko

  • Estimation de l'espace disque requis : 24 Mo

  • Estimation du temps de construction : 1.3 SBU

Dépendances de lxqt-panel

Requises

KDE Frameworks-5.46.0 ou (kguiaddons-5.46.0 for lxqt et solid-5.46.0 for lxqt) (greffon indicateur de clavier), lxqt-globalkeys-0.12.0, libdbusmenu-qt-0.9.3+16.04.20160218, liblxqt-0.12.0, lxmenu-data-0.1.5 et menu-cache-1.1.0

Recommandées

alsa-lib-1.1.6 ou PulseAudio-11.1 (greffon de volume ; voir les explications des commandes ci-dessous s'il n'est pas présent), libstatgrab-0.91 (greffon de surveillance du réseau et de charge CPU), libsysstat-0.4.0 (greffons de surveillance CPU et réseau), libxkbcommon-0.8.0 (greffon indicateur de clavier) et lm_sensors-3.4.0 (greffon de capteurs)

Facultatives

git-2.17.1 (pour télécharger les fichiers de traduction) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisation)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-panel

Installation de lxqt-panel

Installez lxqt-panel en lançant les commandes suivantes : 

sed -e 's:<KF5/KWindowSystem/:<:'               \
    -i plugin-taskbar/lxqttaskgroup.{h,cpp}     &&

mkdir -v build &&
cd       build &&

cmake -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DCMAKE_BUILD_TYPE=Release          \
      -DPULL_TRANSLATIONS=no              \
      ..                                  &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

sed ... plugin-taskbar/lxqttaskgroup.{h,cpp} : Supprime un répertoire codé en dur d'un fichier d'en-tête et permet au système de le trouver lorsque les dépendances du Framework KDE ne sont pas installées dans $LXQT_PREFIX.

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.

-D<plugin>_PLUGIN=no : Désactive $lt;plugin>, où <plugin> peut être CLOCK, COLORPICKER, CPULOAD, DIRECTORYMENU, DESKTOPSWITCH, KBINDICATOR, MAINMENU, MOUNT, QUICKLAUNCH, SENSORS, SHOWDESKTOP, NETWORKMONITOR, SYSSTAT, TASKBAR, STATUSNOTIFIER, TRAY, VOLUME, WORLDCLOCK, ou SPACER,

-DVOLUME_USE_<backend>=no : Désactive le support pour <backend> dans le greffon de volume, où backend est ALSA ou PULSEAUDIO.

Contenu

Répertoires installés: /etc/xdg/lxqt, $LXQT_PREFIX/share/lxqt/lxqt-panel et $LXQT_PREFIX/lib/lxqt-panel
