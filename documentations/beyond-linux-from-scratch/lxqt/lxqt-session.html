lxqt-session-0.12.0

Introduction à lxqt-session

Le paquet lxqt-session contient le gestionnaire de session par défaut de LXQT.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/lxde/lxqt-session/releases/download/0.12.0/lxqt-session-0.12.0.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : b5d10deb6ccc964b14f93c3a6451b069

  • Taille du téléchargement : 131 Ko

  • Estimation de l'espace disque requis : 7.7 Mo

  • Estimation du temps de construction : 0.3 SBU

Dépendances de lxqt-session

Requises

liblxqt-0.12.0, (kwindowsystem-5.46.0 for lxqt ou KDE Frameworks-5.46.0) et Xdg-user-dirs-0.17

Facultatives

git-2.17.1 (pour télécharger les fichiers de traduction) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisation)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/lxqt-session

Installation de lxqt-session

Installez lxqt-session en lançant les commandes suivantes : 

sed -e '/TryExec/s@=@='$LXQT_PREFIX'/bin/@' \
    -i xsession/lxqt.desktop.in &&

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

sed -e '/TryExec/s@=@='$LXQT_PREFIX'/bin/@' -i xsessions/lxqt.desktop.in : Si vous utilisez un gestionnaire d'affichage, le chemin complet de la directive « TryExec » doit être donné pour que le bureau LXQt apparaisse dans la liste des sessions.

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.

Configuration de lxqt-session

Des gestionnaires de fenêtres autres que openbox-3.6.1 peuvent être utilisés, par exemple Xfwm4-4.12.4. Notez que IceWM-1.4.2 n'est pas un remplaçant valide. Fluxbox fonctionne, cependant dans ce contexte (avec lxqt-config-0.12.0) openbox-3.6.1 est meilleur. Le fichier de configuration /etc/xdg/lxqt/windowmanagers.conf a plusieurs exemples de gestionnaires de fenêtres et ceux qui sont installés apparaîtront dans une liste déroulante de lxqt-config-session. Pour ceux qui ne sont pas dans /etc/xdg/lxqt/windowmanagers.conf, vous pouvez utiliser le bouton « recherche » de lxqt-config-session, par exemple pour Fluxbox-1.3.7, en naviguant dans les fichiers système jusqu'à ce que vous choisissiez fluxbox. Autrement, si vous voulez que le gestionnaire de fenêtres de votre choix apparaisse dans la liste, ajoutez-le dans /etc/xdg/lxqt/windowmanagers.conf. Pour Fluxbox-1.3.7, ajoutez la ligne : 

fluxbox/Name=Fluxbox

et il apparaîtra comme une option dans la liste déroulante. Pour changer de gestionnaire de fenêtre, après en avoir sélectionné un, vous devez redémarrer la session.

Descriptions courtes

lxqt-config-session

est un outil de configuration graphique pour les paramètres de lxqt-session.

lxqt-leave

est une boîte de dialogue graphique pour terminer la session.

lxqt-session

est un gestionnaire de session X léger.

startlxqt

utilisé pour démarrer la session graphique pour LXQT.
