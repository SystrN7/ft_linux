pcmanfm-qt-0.12.0

Introduction à pcmanfm-qt

Le paquet pcmanfm-qt est un gestionnaire de fichiers et un gestionnaire d'icônes de bureau (le port Qt de pcmanfm et libfm).

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Dépendances de pcmanfm-qt

Requises

liblxqt-0.12.0, libfm-qt-0.12.0 et lxmenu-data-0.1.5

Recommandées

oxygen-icons5-5.46.0 ou un autre thème d'icônes (en fonction de votre choix, certaines icônes peuvent manquer à plusieurs endroits)

Facultatives

git-2.17.1 (pour télécharger les fichiers de traduction) ou lxqt-l10n-0.12.0 (pour ajouter les fichiers de localisation)

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/pcmanfm-qt

Installation de pcmanfm-qt

Installez pcmanfm-qt en lançant les commandes suivantes : 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DPULL_TRANSLATIONS=no              \
      ..       &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

-DPULL_TRANSLATIONS=no : Supprime le besoin de git-2.17.1, qui serait sinon demandé pour télécharger les fichiers de traduction.
