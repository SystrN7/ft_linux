MuPDF-1.14.0

Introduction à MuPDF

MuPDF est une visionneuse légère de PDF et de XPS.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://www.mupdf.com/downloads/archive/mupdf-1.14.0-source.tar.gz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 98adc2f430cc7900397ab50a478485c5

  • Taille du téléchargement : 53 Mo

  • Estimation de l'espace disque requis : 268 Mo

  • Estimation du temps de construction : 0.1 SBU (Avec parallélisme = 4)

Installation de MuPDF

Installez MuPDF en lançant les commandes suivantes : 

patch -Np1 -i ../mupdf-1.14.0-shared_libs-1.patch &&

USE_SYSTEM_LIBS=yes make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

USE_SYSTEM_LIBS=yes                     \
make prefix=/usr                        \
     build=release                      \
     docdir=/usr/share/doc/mupdf-1.14.0 \
     install                            &&

ln -sfv mupdf-x11 /usr/bin/mupdf        &&
ldconfig

Explication des commandes

ln -sfv mupdf-x11-curl /usr/bin/mupdf  : ce lien symbolique permet de choisir entre mupdf-gl et mupdf-x11 en lançant mupdf.

Contenu

Programme installé: mupdf (lien symbolique), mupdf-gl, mupdf-x11 et mutool
Bibliothèques installées: libmupdf.so, libmupdfthird.so, libmupkcs7.so et libmuthreads.so
Répertoires installés: /usr/include/mupdf, /usr/share/doc/mupdf-1.14.0

Descriptions courtes

mupdf

est un programme pour visualiser des documents PDF, XPS, EPUB et CBZ et divers formats d'image comme PNG, JPEG, GIFF et TIFF.

mupdf-gl

comme mupdf, avec le rendu opengl.

mupdf-x11

comme mapdf, avec le rendu X Window.

libmupdf.so

contient les fonctions de l'API de mupdf.

libmupdfthird.so

contient les bibliothèques tierces compilées pour mupdf.
