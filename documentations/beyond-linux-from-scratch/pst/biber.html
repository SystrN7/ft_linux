biber-2.12

Introduction à biber

Biber est un remplaçant pour BibTex pour les utilisateurs de biblatex, écrit en Perl avec le support complet d'Unicode.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : https://github.com/plk/biber/archive/v2.12.tar.gz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 2bbaffa7af5fedbcede60cfeaf56cd7c

  • Taille du téléchargement : 1.3 Mo

  • Estimation de l'espace disque requis : 36 Mo

  • Estimation du temps de construction : 0.4 SBU dont les tests

Note

L'archive des sources de biber montré en téléchargement avec le bon nom, biber-2.12.tar.gz, en utilisant un navigateur comme Firefox. Si vous préférez utiliser un programme en ligne de commande comme wget, vous obtiendrez normalement v2.12.tar.gz. Pour obtenir ce paquet avec le nom de fichier correct, lancez  : 

wget -c https://github.com/plk/biber/archive/v2.12.tar.gz \
     -O biber-2.12.tar.gz

.

Téléchargement supplémentaire requis

  • La version correspondante de biblatex est un projet séparé - l'archive de tds est pré-emballée pour le déballage dans le répertoire texmf-dist. Téléchargement (HTTP) : http://sourceforge.net/projects/biblatex/files/biblatex-3.12/biblatex-3.12.tds.tgz

Note

Il est possible d'installer toutes les dépendances manquantes automatiquement. Sauf qu'à partir de perl-5.22.0, Module-Build-0.4224 n'est plus une partie du cœur de la distribution. Pour installer automatiquement les dépendances perl restantes vous pouvez d'abord installer ce module en utilisant les automatic installation of perl modules. Vous pouvez ensuite lancer perl ./Build.PL et quand il vous donne la main, devenez utilisateur root et lancez ./Build installdeps

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/biber

Installation de Biber

Installez Biber en lançant les commandes suivantes : 

perl ./Build.PL &&
./Build

Pour tester les résultats, entrez : ./Build test

Maintenant, en tant qu'utilisateur root : 

tar -xf ../biblatex-3.12.tds.tgz -C /opt/texlive/2018/texmf-dist &&
texhash &&
./Build install

Explication des commandes

tar -xf biblatex-3.12.tds.tar.gz -C /opt/texlive/2018/texmf-dist : ceci installe les nouveaux fichiers biblatex par-dessus ceux installés par texlive.

texhash : ceci met à jour les tables de hashage des fichiers (aussi connu comme la base de donnée de noms de fichiers).
