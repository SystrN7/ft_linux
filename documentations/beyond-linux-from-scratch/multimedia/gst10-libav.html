Introduction à GStreamer Libav

Le paquet GStreamer Libav contient les greffons GStreamer pour Libav (un fork de FFmpeg).

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : https://gstreamer.freedesktop.org/src/gst-libav/gst-libav-1.14.4.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 58342db11dbb201a66a62577dcf7bab5

  • Taille du téléchargement : 8.6 Mo

  • Estimation de l'espace disque requis : 634 Mo (avec les tests)

  • Estimation du temps de construction : 1.5 SBU (avec parallélisme = 4 ; avec les tests)

Dépendances de GStreamer Libav

Requises

gst-plugins-base-1.14.4

Recommandées

yasm-1.3.0

Facultatives

Valgrind-3.14.0 et Orc

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/gst10-libav

Installation de GStreamer Libav

Installez GStreamer Libav en lançant les commandes suivantes : 

./configure --prefix=/usr \
            --with-package-name="GStreamer Libav Plugins 1.14.4 BLFS" \
            --with-package-origin="http://www.linuxfromscratch.org/blfs/view/svn/" &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--with-libav-extra-configure="--disable-yasm" : Utilisez ce paramètre si vous n'avez pas installé yasm.
