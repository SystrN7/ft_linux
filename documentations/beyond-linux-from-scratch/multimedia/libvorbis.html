libvorbis-1.3.6

Introduction à libvorbis

Le paquet libvorbis contient un format d'encodage de musique et de son généraliste. C'est utile pour créer (encoder) et lire (décoder) du son dans un format libre (sans brevet).

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de libvorbis

Installez libvorbis en lançant les commandes suivantes : 

./configure --prefix=/usr --disable-static &&
make

Pour tester les résultats lancez : make LIBS=-lm check.

Maintenant, en tant qu'utilisateur root : 

make install &&
install -v -m644 doc/Vorbis* /usr/share/doc/libvorbis-1.3.6

Explication des commandes

--disable-static : Ce paramètre empêche l'installation des versions statiques des bibliothèques.

--enable-docs : Ce paramètre active la construction de la documentation dans des formats autre que celle fournie en HTML.

Contenu
