gstreamer-vaapi-1.14.4

Introduction à gstreamer-vaapi

Le paquet gstreamer-vaapi contient un greffon gstreamer pour l'encodage/décodage de vidéo accélérés par le matériel pour les standards d'encodage prévalents de nos jours (MPEG-2, MPEG-4 ASP/H.263, MPEG-4 AVC/H.264 et VC-1/VMW3).

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Dépendances de gstreamer-vaapi

Requises

gstreamer-1.14.4, gst-plugins-base-1.14.4, gst-plugins-bad-1.14.4 et libva-2.4.0

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/gstreamer10-vaapi

Installation de gstreamer-vaapi

Installez gstreamer-vaapi en lançant les commandes suivantes : 

./configure --prefix=/usr &&
make

Les tests peuvent être lancés manuellement dans le répertoire tests. Ils nécessitent un affichage X Window lancé.

Maintenant, en tant qu'utilisateur root : 

make install
