Introduction à Frei0r-plugins

Frei0r est une API de greffon minimaliste pour les effets vidéo. Remarquez que le 0 dans le nom est un zéro, et non la lettre o en majuscule.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de Frei0r-plugins

Installez Frei0r en lançant les commandes suivantes : 

sed -i 's/CV_RGB/cv::Scalar/' src/filter/facebl0r/facebl0r.cpp &&

mkdir -vp build &&
cd        build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr    \
      -DCMAKE_BUILD_TYPE=Release     \
      -Wno-dev ..                    &&

make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

sed … facebl0r.cpp : Cette commande corrige un problème de construction avec la dépendance opencv actuelle.

-DWITHOUT_GAVL=ON : Cette option désactive la construction des greffons dépendant de gavl.
