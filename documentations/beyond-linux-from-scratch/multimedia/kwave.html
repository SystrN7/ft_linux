Introduction à KWave

Le paquet KWave contient une application d'édition audio basée sur KF5.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://download.kde.org/stable/applications/18.12.2/src/kwave-18.12.2.tar.xz

Dépendances de KWave

Requises

AudioFile-0.3.6, fftw-3.3.8, id3lib-3.8.3 et KDE Frameworks-5.55.0

Recommandées

Installation de KWave

Installez KWave en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
