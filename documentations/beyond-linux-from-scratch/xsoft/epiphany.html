Epiphany-3.30.3

Introduction à Epiphany

Epiphany est un simple mais puissant navigateur web de GNOME destiné aux utilisateurs non-techniciens. Ces principes sont la simplicité et le respect des standards.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Installation de Epiphany

Installez Epiphany en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

meson --prefix=/usr .. &&
ninja

Maintenant, en tant qu'utilisateur root : 

ninja install &&
glib-compile-schemas /usr/share/glib-2.0/schemas

Ce paquet n'a pas de suite de tests fonctionnelle.
