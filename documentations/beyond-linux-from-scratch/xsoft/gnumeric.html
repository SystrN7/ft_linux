Gnumeric-1.12.44

Introduction à Gnumeric

Le paquet Gnumeric contient un programme tableur qui est utile pour de l'analyse mathématique.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : http://ftp.gnome.org/pub/gnome/sources/gnumeric/1.12/gnumeric-1.12.44.tar.xz

  • Téléchargement (FTP) : ftp://ftp.gnome.org/pub/gnome/sources/gnumeric/1.12/gnumeric-1.12.44.tar.xz

  • Somme de contrôle MD5 du téléchargement : 0650750707491b4fcea0eb3010cafbb1

  • Taille du téléchargement : 17 Mo

  • Estimation de l'espace disque requis : 262 Mo (plus 4 Mo pour les tests)

  • Estimation du temps de construction : 1.1 SBU (avec parallélisme = 4 ; plus 3.9 SBU pour les tests)

Dépendances de Gnumeric

Requises

GOffice-0.10.44, itstool-2.0.5 et Rarian-0.8.1

Recommandées à l'exécution

un thème d'icônes comme adwaita-icon-theme-3.30.1, oxygen-icons5-5.55.0 ou gnome-icon-theme-3.12.0, Système X Window et Yelp-3.30.0 pour les fonctions d'aide intégrées.

Facultatives

DConf-0.30.1 (pour empêcher des échecs aux tests), gobject-introspection-1.58.3, PyGObject-3.30.4, Valgrind-3.14.0 (pour certains tests), dblatex (pour les docs PDF), libgda, Mono, pxlib et Psiconv

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/gnumeric

Installation de Gnumeric

Installez Gnumeric en lançant les commandes suivantes : 

sed -i 's/HELP_LINGUAS = cs de es/HELP_LINGUAS = de es/' doc/Makefile.in &&
./configure --prefix=/usr  &&
make

Ce paquet doit être installé avant d'exécuter la suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Pour lancer les tests, tapez make check. Les tests doivent être lancés dans une session X active, et DConf-0.30.1 devrait être installé pour éviter certains échecs. Neuf tests, dont la suite valgrind entière, sont connus pour échouer.

Explication des commandes

--enable-pdfdocs : utilisez ce paramètre si vous avez installé dblatex et voulez créer des docs PDF.

Contenu

Programmes installés: gnumeric (lien symbolique), gnumeric-1.12.44, ssconvert, ssdiff, ssgrep et ssindex
Bibliothèques installées: libspreadsheet.so
Répertoires installés: /usr/include/libspreadsheet-1.12, /usr/lib/gnumeric, /usr/lib/goffice/0.10/plugins/gnumeric, /usr/share/gnumeric, /usr/share/help/{C,cs,de,es}/gnumeric et /usr/share/pixmaps/gnumeric

Descriptions courtes

gnumeric

est un lien symbolique vers gnumeric-1.12.44

gnumeric-1.12.44

est l'application tableur de GNOME.

ssconvert

est un outil en ligne de commande utilisé pour convertir des fichiers de feuilles de calcul vers différents formats de fichiers de feuilles de calcul.

ssdiff

est un outil en ligne de commande pour comparer deux feuilles de calcul

ssgrep

est un utilitaire en ligne de commande pour rechercher des feuilles de calcul pour les chaînes.

ssindex

est un outil en ligne de commande pour générer des données d'index pour des fichiers de feuille de calcul.

libspreadsheet.so

contient les fonctions de l'API de gnumeric.
