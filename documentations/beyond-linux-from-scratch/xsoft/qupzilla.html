QupZilla-2.2.6

Introduction à QupZilla

QupZilla est un navigateur léger et complet basé sur QtWebEngine, à l'origine conçu pour l'éducation.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.2.

Installation de QupZilla

Installez QupZilla en lançant les commandes suivantes : 

export QUPZILLA_PREFIX=/usr &&
qmake                       &&
make

La commande export est utilisée pour configurer la construction. Autrement, vous pouvez ajouter la variable d'environnement à src/defines.pri.

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Si desktop-file-utils-0.23 et xdg-utils-1.1.3 sont installé, lancez en tant qu'utilisateur root : 

xdg-icon-resource forceupdate --theme hicolor &&
update-desktop-database -q

Explication des commandes

Note

Si vous souhaitez utiliser l'une de ces variables, elles doivent être exportées. Vous pouvez les ajouter au bloc export dans les instructions principales. Autrement, plutôt que de les exporter, elles peuvent être ajoutées, une par ligne, dans src/defines.pri.

CONFIG += debug : Utilisez-la si vous voulez construire QupZilla avec les symboles de débogage.

GNOME_INTEGRATION=true : Active l'intégration Gnome, ce qui signifie la construction du greffon Gnome-Keyring Password, qui fournit un support pour sauvegarder les mots de passe dans Gnome-Keyring.
