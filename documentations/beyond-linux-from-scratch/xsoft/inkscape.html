Inkscape-0.92.4

Introduction à Inkscape

Inkscape est un éditeur graphique d'images vectorielles évolutives. Il sert à créer, voir et modifier des images SVG.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Somme de contrôle MD5 du téléchargement : ac30f6d5747fd9c620c00dad500f414f

Dépendances de Inkscape

Requises

Boost-1.69.0, GC-8.0.2, Gsl-2.5, Gtkmm-2.24.5 (ou Gtkmm-3.24.0 et gnome docking library pour la construction gtk+-3 expérimentale), libxslt-1.1.33, Poppler-0.74.0, popt-1.16 et Wget-1.20.1 (pour télécharger les dépendances des tests)

Recommandées

ImageMagick-6.9.10-27 libraries, Little CMS-2.9 or Little CMS-1.19, Potrace-1.15 (pour l'outil bucket-fill), lxml-4.3.1 et Scour-0.37 (tous les deux à l'exécution, pour Save As Optimized SVG).

Facultatives

Aspell-0.60.6.1, dbus-1.12.12 (pour lancer inkscape depuis des scritps), Doxygen-1.8.15, libcdr, libvisio, libwpg (ou libwpd) et NumPy (à l'exécution pour certaines extensions)

Installation de Inkscape

Assurez-vous que les bibliothèques de ImageMagick-6.9.10-27 libraries sont utilisables : 

patch -Np1 -i ../inkscape-0.92.4-use_versioned_ImageMagick6-1.patch

Si vous souhaitez lancer la suite de tests, téléchargez les versions requises de googletest et googlemock en lançant : 

bash download-gtest.sh

Installez Inkscape en lançant les commandes suivantes : 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DCMAKE_BUILD_TYPE=Release  \
      ..                          &&
make

Pour tester les résultats lancez : make check.

Maintenant, en tant qu'utilisateur root : 

make install                      &&
rm -v /usr/lib/inkscape/lib*_LIB.a

Note

Ce paquet installe des fichiers d'icônes dans la hiérarchie /usr/share/icons/hicolor et des fichiers du bureau dans la hiérarchie /usr/share/applications. Vous pouvez améliorer les performances du système et l'utilisation de la mémoire en mettant à jour /usr/share/icons/hicolor/index.theme et /usr/share/applications/mimeinfo.cache. Pour réaliser la mise à jour vous devez avoir installé GTK+-2.24.32 ou GTK+-3.24.5 (pour le cache des icônes) et desktop-file-utils-0.23 (pour le cache du bureau) et taper la commande suivante en tant qu'utilisateur root : 

gtk-update-icon-cache &&
update-desktop-database

Explication des commandes

-DCMAKE_BUILD_TYPE=Release : Ce paramètre est utilisé pour construire la bibliothèque de production sans aucun « assert » dans le code.

rm -v /usr/lib/inkscape/lib*_LIB.a : La conversion du paquet pour utiliser cmake a fait que ces bibliothèques sont installées, mais elles ne sont pas utilisables (il n'y a pas de fichiers d'en-têtes pour savoir ce qu'elles contiennent). Donc supprimez-les.

-DWITH_DBUS=ON : utilisez cela si vous souhaitez utiliser inkscape dans des scripts interactifs qui manipules des images.
