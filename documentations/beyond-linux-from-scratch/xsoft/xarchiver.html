xarchiver-0.5.4

Introduction à xarchiver

XArchiver est un gestionnaire d'archives en GTK+ qui gère le tar, xz, bzip2, gzip, zip, 7z, rar, lzo et bien d'autres formats d'archive.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Note

xarchiver est seulement une interface en ligne de commande ou graphique avec les utilitaires d'archivage comme tar et zip.

Installation de xarchiver

Installez xarchiver en lançant les commandes suivantes : 

patch -Np1 -i ../xarchiver-0.5.4-fixes-1.patch &&

./autogen.sh --prefix=/usr               \
             --libexecdir=/usr/lib/xfce4 \
             --disable-gtk3              \
             --docdir=/usr/share/doc/xarchiver-0.5.4 &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make DOCDIR=/usr/share/doc/xarchiver-0.5.4 install

Si desktop-file-utils-0.23 et hicolor-icon-theme-0.17 sont installés, lancez, en tant qu'utilisateur root : 

update-desktop-database &&
gtk-update-icon-cache -t -f --include-image-data /usr/share/icons/hicolor

Explication des commandes

--libexecdir=/usr/lib/xfce4 : Ce paramètre corrige l'emplacement de thunar-archive-plugin, pour qu'il fonctionne si Thunar-1.8.4 est installé.

---disable-gtk3 : Ce paramètre choisit de construire avec GTK+2. Par défaut, on a --enable-gtk3. Supprimez-le ou remplacez-le si vous souhaitez construire avec GTK+3.
