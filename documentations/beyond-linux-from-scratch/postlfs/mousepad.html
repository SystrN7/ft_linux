Mousepad-0.4.1

Introduction à Mousepad

Mousepad est un éditeur de textes simple utilisant GTK+ 2 pour l'environnement de bureau Xfce.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

Dépendances de Mousepad

Requises

gtksourceview-3.24.9 (facultativement, il peut être construit avec gtksourceview-2)

Facultatives

DConf-0.30.1 (exécution) et dbus-glib-0.110

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/mousepad

Installation de Mousepad

Installez Mousepad en lançant les commandes suivantes : 

./configure --prefix=/usr --enable-keyfile-settings &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

--enable-keyfile-settings : Utilisez le moteur keyfile de GSettings plutôt que DConf-0.30.1 par défaut.
