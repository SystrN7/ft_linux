libcap-2.26 avec PAM

Introduction à libcap avec PAM

Le paquet libcap est installé par LFS, mais si le support de Linux-PAM est souhaité, le module PAM doit être construit (après l'installation de Linux-PAM).

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (HTTP) : https://www.kernel.org/pub/linux/libs/security/linux-privs/libcap2/libcap-2.26.tar.xz

  • Téléchargement (FTP) :

  • Somme de contrôle MD5 du téléchargement : 968ac4d42a1a71754313527be2ab5df3

  • Taille du téléchargement : 68 Ko

  • Estimation de l'espace disque requis : 616 Ko

  • Estimation du temps de construction : moins de 0.1 SBU

Dépendances de libcap

Requises

Linux-PAM-1.3.0

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/libcap

Installation de libcap

Installez libcap en lançant les commandes suivantes : 

make -C pam_cap

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

install -v -m755 pam_cap/pam_cap.so /lib/security &&
install -v -m644 pam_cap/capability.conf /etc/security
