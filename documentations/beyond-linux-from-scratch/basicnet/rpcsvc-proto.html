rpcsvc-proto-1.4

Introduction à rpcsvc-proto

Le paquet rpcsvc-proto contient les fichiers et les en-têtes rcpsvc protocol.x, précédemment inclus dans glibc et qui ne sont pas inclus dans le paquet libtirpc-1.1.4 de remplacement, ainsi que le programme rpcgen.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Informations sur le paquet

  • Téléchargement (FTP) :

  • Estimation du temps de construction : moins de 0.1 SBU

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/rpcsvc-proto

Installation de rpcsvc-proto

Installez rpcsvc-proto en lançant les commandes suivantes : 

./configure --sysconfdir=/etc &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install
