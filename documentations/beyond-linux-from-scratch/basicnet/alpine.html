Alpine-2.21

Introduction à Alpine

Alpine est un client de courriel textuel développé par l'université de Washington.

Ce paquet est connu pour se construire correctement sur une plateforme LFS-8.4.

Dépendances de Alpine

Facultatives

OpenLDAP-2.4.47, MIT Kerberos V5-1.17, Aspell-0.60.6.1, Tcl-8.6.9 et Linux-PAM-1.3.0

Notes utilisateur : http://wiki.linuxfromscratch.org/blfs/wiki/alpine

Installation de Alpine

Installez Alpine en lançant les commandes suivantes : 

LIBS+="-lcrypto" ./configure --prefix=/usr       \
                             --sysconfdir=/etc   \
                             --without-ldap      \
                             --without-krb5      \
                             --without-pam       \
                             --without-tcl       \
                             --with-ssl-dir=/usr \
                             --with-passfile=.pine-passfile &&
make

Ce paquet n'est pas fourni avec une suite de tests.

Maintenant, en tant qu'utilisateur root : 

make install

Explication des commandes

LIBS+="-lcrypto" : Active la liaison à openssl. Des drapeaux de l'éditeur de lien supplémentaires sont requis pour Linux-PAM-1.3.0, -lpam et pour MIT Kerberos V5-1.17, -lkrb5.

--without-ldap : Désactive le support de LDAP.

--without-krb5 : Désactive le support de Kerberos.

--without-pam : Désactive le support de Linux-PAM.

--without-tcl : Désactive le support de TCL.

--with-ssl-dir=/usr : Initialise le chemin de la racine vers les bibliothèques et les fichiers d'en-tête de OpenSSL.

--with-passfile=.pine-passfile : Initialise le fichier de cache de mots de passe.

--without-ssl : Désactive le support d'OpenSSL.

Configuration de Alpine

Fichiers de configuration

~/.pinerc

Informations sur la configuration

Il n'est pas nécessaire de modifier manuellement aucun fichier de configuration pour utiliser le client de courriel Alpine. Les utilisateurs peuvent configurer Alpine avec le menu de configuration graphique, qui enregistre les changements dans ~/.pinerc.

Descriptions courtes

alpine

est le courrielleur Alpine.

pico

est un éditeur de texte autonome similaire au compositeur de message d'Alpine.

pilot

est un navigateur du système de fichiers autonome.

rpdump

est un utilitaire pour télécharger un pinerc ou un carnet d'adresses sur la machine locale.

rpload

est un utilitaire pour téléverser un pinerc ou un carnet d'adresseslocal version un serveur IMAP.
